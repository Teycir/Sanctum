Creating a Vault

1 Configure IPFS Provider
Choose between Pinata or Filebase (both free). Enter your API credentials:
  • Pinata: JWT token from pinata.cloud
  • Filebase: Access Key + Secret Key from filebase.com
2 Choose Vault Mode
  • Simple Vault: Basic encrypted storage without deniability
  • Duress Hidden: Two layers (decoy + hidden) for plausible deniability
3 Add Content
  • Decoy Layer: Innocent content shown without passphrase (optional but recommended)
  • Hidden Layer: Your real secrets (text or .zip/.rar file up to 25MB)
4 Set Passphrase
Create a strong passphrase for the hidden layer. Use 6+ Diceware words or a memorable phrase. Never store digitally.
5 Create & Save Link
Click "Create Vault" and save the generated link. This link + passphrase = access to hidden layer.
‚ö†Ô∏è Critical: If you lose the vault link, your data is lost forever. Save it in a password manager or print the QR code.

Unlocking a Vault

1 Open Vault Link
Navigate to your saved vault link in a browser.
2 Enter Passphrase
  • No passphrase: Shows decoy layer
  • Correct passphrase: Shows hidden layer
  • Wrong passphrase: Shows decoy layer (by design)
3 Access Content
Download files or copy text. Clipboard auto-clears after 60 seconds for security.

Security Features

Auto-Lock
Vault automatically locks after 5 minutes of inactivity. Re-enter passphrase to unlock.
Panic Key
Double-press Escape to instantly lock the vault. Use in emergency situations.
Secure Clipboard
Copied content automatically clears from clipboard after 60 seconds to prevent data leakage.

Best Practices

  • Save vault links securely - Use a password manager (KeePassXC, Bitwarden) or print QR codes
  • Use strong passphrases - 6+ Diceware words (e.g., "correct-horse-battery-staple-mountain-river")
  • Make decoy content believable - Add realistic files/wallet with $50-500 to maintain plausibility
  • Test before trusting - Verify both decoy and hidden layers unlock correctly before relying on them
  • Use Tor Browser for high-risk scenarios - Hides your IP from IPFS gateways and prevents tracking
